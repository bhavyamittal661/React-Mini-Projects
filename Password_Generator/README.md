## Features

- Generate random passwords of any length
- Option to include/exclude numbers
- Option to include/exclude symbols
- Responsive and mobile-friendly design
- Easy to copy and extend

---

## How to Run

1. Clone the repository or navigate inside your main repo:

```bash
cd Password_Generator
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the app in your browser at:

http://localhost:5173

---

## Usage

1. Adjust the password length using the slider.
2. Toggle Include Numbers and Include Symbols options as needed.
3. Click Generate Password.
4. Copy the password manually to use it in your applications.
